# Capstone_Dinner_Party
Capstone project - Event planning App


Main scripts:

TS compile & watch
```bash
npm run compile:watch
```

Webpack build & watch
```bash
npm run start:dev-build
```

Node-dev & watch
```bash
npm run start:dev-server
```

Sync database
```bash
npm run syncDB
```

Seed data
```bash
npm run seed
```
